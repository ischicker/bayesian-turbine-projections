# Data

## Raw Data (`raw/`)

Raw data files are not tracked in this repository due to size and licensing.
Download from the original sources:

- **USWTDB** (US): https://eerscmap.usgs.gov/uswtdb/
- **MaStR** (Germany): https://www.marktstammdatenregister.de/
- **GOWIRES**: https://doi.org/10.5281/zenodo.18768952
- **Austrian data** (IGW/UVP): Available from the authors upon reasonable
  request and with permission of the data providers.

Place downloaded files in the corresponding subdirectories under `raw/`.

## Processed Data (`processed/`)

Generated by `scripts/02_prepare_data.py`. The following aggregated files are
included in the repository for convenience:

| File | Description | Public |
|------|-------------|--------|
| `at_annual.csv` | Austrian annual medians and IQRs | Yes |
| `de_annual.csv` | German annual medians and IQRs | Yes |
| `us_annual.csv` | US annual medians and IQRs | Yes |
| `gowires_wind_climate.csv` | GOWIRES Weibull parameters for reference sites | Yes |
| `gowires_reference_neighbors.csv` | GOWIRES neighbor metadata | Yes |

The following files are generated by the pipeline but not tracked (too large or
confidential):

| File | Description | Why excluded |
|------|-------------|--------------|
| `at_clean.csv` | Austrian turbine-level observations | Confidential |
| `de_clean.csv` | German turbine-level observations | Large (3.8 MB) |
| `us_clean.csv` | US turbine-level observations | Large (6.6 MB) |
| `energy_yield_propagation_samples.csv` | AEP posterior samples | Large (2.9 MB) |
